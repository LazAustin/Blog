This is my personal blog. I'm building this as a way to learn MangoDB, Express, React, and Node (the MERN stack). I'm also learning MaterialUI and a couple of API's that I'm undecided on yet. It will also include some password security which I have not implemented in an original project before. This is actually really fun. I used to have a blog in college and a column in the college newspaper. Now I get to combine my passion for reading/writing with coding. I've watched probably a dozen tutorials on MERN, MaterialUI, blogs, etc but none were quite what I'm wanting so like a true programmer, I'm creating my own thing, combining and mixing features I liked from other sites and also digging into documentation to add features that I haven't learned or seen in any tutorials. It will be a dynamic, multi-page blog with it's own post editor, a user registration/password system, a category system, RSS feeds, etc.
